# OTUS Administrator Linux. Professional course 
## HW10 Working with processes
Дописать обработчики сигналов в прилагаемом скрипте, оттестировать, приложить сам скрипт, инструкции по использованию

#### Требования
- Python 3.6+
#### Как запустить
```
python3 fork.py
```

#### Как работает
1. Основной процесс форкается и ждет выполнения дочернего процесса
2. Дочерний процесс форкается несколько раз и собирает статусы выполнения детей
3. Последнее поколение, делает вид полезной нагрузки (на самом деле спит) и возвращает случайный статус
